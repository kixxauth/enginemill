Welcome to Enginemill.

Enginemill is a Node.js web development framework. The goal is to codify some opinions about how to structure a Node.js system and provide some tools to make the development of your systems easier and more fun.

Only The Basics

  • Includes Bluebird for Promises.
  • Includes Lodash (Underscore) too.
  • Includes Filepath to work with the filesystem.
  • Parses command line options with Yargs.
  • Serially loads plugins you define and kicks off your app only when they have all loaded.
  • Includes a handy Promisified request wrapper for making HTTP requests.
  • Supports CoffeeScript out of the box, which is nice for config and plugin initialization files.
